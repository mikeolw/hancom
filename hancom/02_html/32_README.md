# 개발자도구 (F12) 🔧
## 화면 뒤를 들여다보는 꿀팁

> 브라우저에 숨어 있는 개발자도구 — 화면 뒤 HTML·CSS가 다 보이고, 값을 잠깐 바꿔 미리보기도 됨
> 
> 에러를 잡는 실전 도구 **(별도 설치 X, 크롬·엣지 기본 내장)**

---

## 📖 여는 법

| 방법 | 단축키 |
|------|--------|
| **단축키** | F12 (Mac: Cmd + Opt + I) |
| **마우스** | 화면 아무 곳 우클릭 → 검사 |

---

## 💡 꿀팁 — 이렇게 써먹기

### 1️⃣ 요소 검사하기
화살표(검사) 아이콘을 클릭 → 화면 요소에 마우스를 올리면 그 태그가 무엇인지 즉시 표시된다.

### 2️⃣ HTML 구조 확인하기
**Elements 탭**에서 페이지의 HTML 구조를 볼 수 있다.
- 줄을 클릭하면 화면에서 그 부분이 반짝 표시됨

### 3️⃣ 디자인 미리보기
글자나 색 값을 더블클릭해 바꿔본다.
- 변경사항이 화면에 바로 반영됨
- 디자인을 실시간으로 미리볼 수 있음

### 4️⃣ 에러 확인하기
**Console 탭**에서 빨간 글씨를 확인하면 에러 메시지를 볼 수 있다.
- 어디가 틀렸는지에 대한 힌트를 얻을 수 있음

---

## ⚠️ 주의사항

> 여기서 바꾼 건 **화면에서만 잠깐 보이는 것**이다.
> 
> - 내 파일은 안 바뀜
> - 새로고침하면 원래대로 돌아감
> - **마음껏 실험해도 안전함** ✅
